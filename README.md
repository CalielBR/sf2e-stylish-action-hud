## ✨ Principais Funcionalidades

- 🔗 **Otimização Tática**: Adaptado para combate de alta tecnologia (motor SF2e / PF2e).
- ⚡ **HUD Automatizado**: Preenche automaticamente com ações, perícias, magias e equipamentos.
- 🧼 **Interface Limpa**: Design organizado para evitar poluição visual.
- 📏 **Submenus Ajustados**: Largura otimizada para nomes longos de habilidades.
- 🚀 **Plug and Play**: Funciona imediatamente sem necessidade de configuração.
- 🧩 **Compatibilidade**: Totalmente compatível com Foundry VTT V12 e V13.

---

## 📦 Requisitos

Este módulo requer os seguintes pacotes instalados:
- **Sistema**: Starfinder 2nd Edition (v0.0.11)
- **Módulo**: [Stylish Action HUD](https://github.com/glitchsmith/stylish-action-hud)

---

## 🚀 Instalação

1. Abra o Foundry VTT.
2. Vá na aba **Add-on Modules**.
3. Clique em **Install Module**.
4. Cole a URL do Manifesto:
https://github.com/CalielBR/sf2e-stylish-action-hud/releases/latest/download/module.json


---

## 🧠 Como Funciona

Este módulo adapta o Stylish Action HUD para o Starfinder 2e, organizando:

- Ações  
- Magias  
- Perícias  
- Equipamentos  

👉 Tudo otimizado para caber na tela sem poluição visual.
