## ✨ Fonctionnalités principales

- 🔗 **Optimisation tactique** : Conçu pour le combat high-tech (moteur SF2e / PF2e).  
- ⚡ **HUD automatisé** : Remplit automatiquement actions, compétences, sorts et équipements.  
- 🧼 **Interface claire** : Organisation pensée pour éviter la surcharge visuelle.  
- 📏 **Sous-menus ajustés** : Largeur optimisée pour les noms longs.  
- 🚀 **Plug and play** : Fonctionne immédiatement sans configuration.  
- 🧩 **Compatibilité** : Compatible avec Foundry VTT V12 et V13.  

---

## 📦 Prérequis

Ce module nécessite :

- **Système** : Starfinder 2nd Edition (v0.0.11)  
- **Module** : [Stylish Action HUD](https://github.com/glitchsmith/stylish-action-hud)

---

## 🚀 Installation

1. Ouvrez Foundry VTT  
2. Allez dans **Add-on Modules**  
3. Cliquez sur **Install Module**  
4. Collez l’URL du manifest :

https://github.com/CalielBR/sf2e-stylish-action-hud/releases/latest/download/module.json

---

## 🧠 Fonctionnement

Ce module adapte Stylish Action HUD pour Starfinder 2e en organisant :

- Actions  
- Sorts  
- Compétences  
- Équipement  

👉 Tout est optimisé pour une utilisation claire et efficace.
